# InvoiceStudio - マーケットプレイス用アプリ説明

**最終更新**: 2024年

## 日本語版

### アプリ名
**InvoiceStudio**

### 短い説明（1-2行）
Monday.comのボードデータからプロフェッショナルな請求書を簡単に作成・編集・ダウンロードできるアプリです。3ステップのシンプルなワークフローで、誰でもすぐに請求書を作成できます。

### 詳細説明

**InvoiceStudio**は、Monday.comのボードデータを活用して、プロフェッショナルな請求書を簡単に作成できるアプリケーションです。

#### 主な機能

✅ **3ステップワークフロー**
- アイテム選択 → 請求書編集 → ダウンロード
- 直感的な操作で誰でも簡単に請求書を作成

✅ **Monday.comボードとの連携**
- ボードのアイテムから自動で請求先情報を取得
- フィールドマッピング機能で、ボードのカラムと請求書項目を自由にマッピング
- サブアイテムを明細として自動変換

✅ **多言語・多通貨対応**
- 日本語・英語・スペイン語に対応
- JPY/USD/EUR/GBP/CNYの5通貨に対応

✅ **3つのテンプレート**
- モダン：シンプルで洗練されたデザイン
- クラシック：伝統的な請求書スタイル
- ミニマル：シンプルで読みやすいデザイン

✅ **カスタマイズ機能**
- テンプレートカラーの変更
- 会社ロゴ・署名画像の追加
- 発行元情報・振込先情報のテンプレート保存
- リアルタイムプレビュー

✅ **セキュリティ**
- データはすべてブラウザ内で処理（サーバーに保存されません）
- Monday.comの認証を使用した安全なデータ取得

#### 使用例

- **プロジェクト管理**: プロジェクトの進捗に応じて請求書を自動生成
- **時間追跡**: 時間追跡データから請求書を作成
- **在庫管理**: 販売データから請求書を生成
- **サービス提供**: サービス提供内容を請求書として出力

#### ターゲットユーザー

- フリーランス・個人事業主
- 中小企業の経理担当者
- プロジェクトマネージャー
- サービス提供事業者

## English Version

### App Name
**InvoiceStudio**

### Short Description (1-2 lines)
Create, edit, and download professional invoices from Monday.com board data with ease. A simple 3-step workflow that makes invoice creation accessible to everyone.

### Detailed Description

**InvoiceStudio** is an application that makes it easy to create professional invoices from Monday.com board data.

#### Key Features

✅ **3-Step Workflow**
- Select Item → Edit Invoice → Download
- Intuitive operation that makes invoice creation simple for anyone

✅ **Monday.com Board Integration**
- Automatically retrieve billing information from board items
- Field mapping feature to freely map board columns to invoice fields
- Automatically convert subitems to line items

✅ **Multi-Language & Multi-Currency Support**
- Supports Japanese, English, and Spanish
- Supports 5 currencies: JPY/USD/EUR/GBP/CNY

✅ **3 Templates**
- Modern: Simple and refined design
- Classic: Traditional invoice style
- Minimal: Simple and readable design

✅ **Customization Features**
- Change template colors
- Add company logo and signature images
- Save templates for issuer and payment information
- Real-time preview

✅ **Security**
- All data is processed in the browser (not stored on servers)
- Secure data retrieval using Monday.com authentication

#### Use Cases

- **Project Management**: Automatically generate invoices based on project progress
- **Time Tracking**: Create invoices from time tracking data
- **Inventory Management**: Generate invoices from sales data
- **Service Provision**: Output service details as invoices

#### Target Users

- Freelancers and sole proprietors
- Accounting staff at small and medium-sized enterprises
- Project managers
- Service providers

## Spanish Version

### Nombre de la Aplicación
**InvoiceStudio**

### Descripción Corta (1-2 líneas)
Cree, edite y descargue facturas profesionales desde los datos del tablero de Monday.com con facilidad. Un flujo de trabajo simple de 3 pasos que hace que la creación de facturas sea accesible para todos.

### Descripción Detallada

**InvoiceStudio** es una aplicación que facilita la creación de facturas profesionales desde los datos del tablero de Monday.com.

#### Características Principales

✅ **Flujo de Trabajo de 3 Pasos**
- Seleccionar Artículo → Editar Factura → Descargar
- Operación intuitiva que hace que la creación de facturas sea simple para cualquiera

✅ **Integración con Tablero de Monday.com**
- Recuperar automáticamente la información de facturación de los artículos del tablero
- Función de mapeo de campos para mapear libremente las columnas del tablero a los campos de factura
- Convertir automáticamente los subartículos en artículos de línea

✅ **Soporte Multiidioma y Multimoneda**
- Soporta japonés, inglés y español
- Soporta 5 monedas: JPY/USD/EUR/GBP/CNY

✅ **3 Plantillas**
- Moderna: Diseño simple y refinado
- Clásica: Estilo de factura tradicional
- Minimalista: Diseño simple y legible

✅ **Características de Personalización**
- Cambiar colores de plantilla
- Agregar logo de empresa e imágenes de firma
- Guardar plantillas para información del emisor y pago
- Vista previa en tiempo real

✅ **Seguridad**
- Todos los datos se procesan en el navegador (no se almacenan en servidores)
- Recuperación segura de datos usando autenticación de Monday.com

#### Casos de Uso

- **Gestión de Proyectos**: Generar automáticamente facturas basadas en el progreso del proyecto
- **Seguimiento de Tiempo**: Crear facturas desde datos de seguimiento de tiempo
- **Gestión de Inventario**: Generar facturas desde datos de ventas
- **Prestación de Servicios**: Generar facturas desde detalles de servicios

#### Usuarios Objetivo

- Freelancers y trabajadores autónomos
- Personal de contabilidad en pequeñas y medianas empresas
- Gerentes de proyecto
- Proveedores de servicios
