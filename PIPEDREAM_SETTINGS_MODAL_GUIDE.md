# PipeDreamのProject Settingsモーダル内でビルド設定を探す方法

**作成日**: 2024年

## 現在の状況

「Settings」をクリックすると「Project Settings」モーダルが開きます。

このモーダルには以下が表示されています：
- プロジェクト名
- Project ID
- GitHub連携設定

**ビルド設定は、このモーダル内の別のタブやセクションにある可能性があります。**

---

## 🔍 Project Settingsモーダル内を確認

### 確認すべき項目

1. **モーダル内にタブがあるか**
   - モーダルの上部や下部にタブがあるか確認
   - 例：「General」「Build」「Deploy」「Environment Variables」など

2. **スクロールして下を見る**
   - モーダルをスクロールして、下にビルド設定があるか確認

3. **セクションを展開する**
   - 「Build Settings」「Deploy Settings」などのセクションがあるか確認
   - セクションが折りたたまれている場合は展開する

---

## 🔍 別の場所を確認

### 方法1: Connectタブを確認

「Connect」タブにデプロイ設定がある可能性があります。

1. プロジェクトの左側パネルで**「Connect」**をクリック
2. デプロイ先の設定を確認
3. ビルド設定があるか確認

---

### 方法2: デプロイ先の設定を確認

PipeDreamで静的サイトをホスティングする場合、デプロイ先の設定が必要です。

#### 確認手順

1. **「Connect」タブを開く**
2. **デプロイ先を確認**
   - 「Deploy」ボタンがあるか
   - 「Hosting」セクションがあるか
   - 「Static Site」セクションがあるか

---

### 方法3: プロジェクトタイプを確認

PipeDreamでは、プロジェクトタイプによって設定項目が異なります。

#### 確認手順

1. **Project Settingsモーダルを開く**
2. **プロジェクトタイプを確認**
   - 「Static Site」タイプか
   - 「Web App」タイプか
   - その他のタイプか

3. **プロジェクトタイプに応じた設定を確認**

---

## 💡 PipeDreamの静的サイトホスティングについて

### 可能性1: PipeDreamが静的サイトを直接ホスティングする場合

設定項目（通常）:
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Node Version**: 18以上（推奨）

設定場所の候補:
1. Project Settingsモーダル内の「Build」タブ
2. Project Settingsモーダル内の「Deploy」タブ
3. 「Connect」タブ内のデプロイ設定
4. 別の「Hosting」タブ

---

### 可能性2: PipeDreamが外部サービスにデプロイする場合

PipeDreamがVercel、Netlifyなどの外部サービスにデプロイする場合：

1. **「Connect」タブを開く**
2. **デプロイ先を選択**（例：Vercel、Netlify）
3. **デプロイ先の設定画面でビルド設定を確認**

---

## 🆘 設定が見つからない場合

### 確認すべきこと

1. **PipeDreamのプラン**
   - 静的サイトホスティング機能が利用可能か確認
   - プランによって機能が異なる場合があります

2. **プロジェクトの作成方法**
   - プロジェクトが「Static Site」タイプとして作成されているか確認
   - プロジェクトタイプによって設定項目が異なります

3. **PipeDreamのドキュメント**
   - 「Static Site Hosting」で検索
   - 「Build Settings」で検索
   - 「Deploy」で検索

---

## 📝 次のステップ

### ステップ1: Project Settingsモーダル内を詳しく確認

1. **モーダルをスクロール**して、下にビルド設定があるか確認
2. **モーダル内にタブがあるか**確認
3. **セクションを展開**して、ビルド設定があるか確認

### ステップ2: Connectタブを確認

1. プロジェクトの左側パネルで**「Connect」**をクリック
2. デプロイ設定を確認

### ステップ3: 画面のスクリーンショットを共有

設定が見つからない場合は、以下を共有してください：

1. **Project Settingsモーダルの全体**（スクロールした状態も）
2. **Connectタブの画面**
3. **左側パネルの全体**

---

## 🔍 代替案: 手動デプロイ

もしPipeDreamにビルド設定の画面がない場合：

### 方法1: ローカルでビルドしてアップロード

1. ローカルでビルド:
   ```bash
   npm run build
   ```

2. `dist/`フォルダの内容をPipeDreamにアップロード
   - 「File Store」タブを使用
   - または手動アップロード機能を使用

### 方法2: GitHub Actionsを使用

1. GitHub Actionsでビルド
2. ビルド結果をPipeDreamにデプロイ
